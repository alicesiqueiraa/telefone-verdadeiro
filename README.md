# Telefone Verdadeiro!

## Como rodar o WebApp

Instale as dependencias com

`npm install`

Vá até a pasta api e execute

`npx prisma db push`

Após isto execute na pasta api

`npm run start`

E tambem execute na pasta app

`npm run dev`

Acesse o cliente através de

http://localhost:3080
